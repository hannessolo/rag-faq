| ## Directions  | ### Directions                                                                                                                                                                        |
|                |                                                                                                                                                                                       |
|                | 1\. Place 2 scoops (approximately 50g) of powder into a mixing bowl and add 100ml of water or milk and stir until fully mixed.                                                        |
|                |                                                                                                                                                                                       |
|                | 2\. Heat a little oil in a non-stick frying pan.                                                                                                                                      |
|                |                                                                                                                                                                                       |
|                | 3\. Using a spoon, drop a large spoonful of the mixture into the pan and cook on a medium high heat for approximately 3-4 minutes on each side until lightly browned.                 |
