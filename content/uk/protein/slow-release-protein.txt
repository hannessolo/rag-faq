## Slow Release Protein FAQS

+-----------------------------------------------------------------------------------------------------------------------+
| Accordion                                                                                                             |
+-----------------------------------------------------------+-----------------------------------------------------------+
| What are slow release protein supplements?                | Slow-release protein supplements are often an             |
|                                                           | afterthought in training. Trainers take shakes and use    |
|                                                           | our fast-release protein powders before and during a      |
|                                                           | workout, but slow-release protein products ensure you can |
|                                                           | still absorb protein, even if you’re not eating it -      |
|                                                           | perfect for a pre-bedtime boost.                          |
+-----------------------------------------------------------+-----------------------------------------------------------+
| What are the benefits of slow release protein powders?    | Slow-release protein powders are a simple way of ensuring |
|                                                           | that your body is getting an adequate amount of protein   |
|                                                           | in, even as you rest. This is what makes it a popular     |
|                                                           | pre-bedtime supplement. It’s the perfect option if you’re |
|                                                           | looking to bulk up or maintain your weight.               |
+-----------------------------------------------------------+-----------------------------------------------------------+
| What is the difference between slow release and fast      | Like carbs, protein comes in slow-release and             |
| release protein?                                          | fast-release forms. We’re usually familiar with           |
|                                                           | fast-release - that Whey Isolate we take before our       |
|                                                           | workout, the Beef Protein Isolate we sip at lunch. These  |
|                                                           | act fast - like most powders, they’re absorbed quickly    |
|                                                           | and boost our protein intake there and then. Slow-release |
|                                                           | proteins take more time to digest, but their work is      |
|                                                           | sustained over a longer period of time, perfect just      |
|                                                           | before bedtime for example.                               |
+-----------------------------------------------------------+-----------------------------------------------------------+
| Is it OK to take slow release protein supplements every   | Slow-release protein supplements are designed to be taken |
| day?                                                      | as part of a healthy, balanced diet. The benefits of this |
|                                                           | slow-release are clear, but some products, such as our    |
|                                                           | flapjacks, contain a greater sugar content, so please     |
|                                                           | bear this in mind.                                        |
+-----------------------------------------------------------+-----------------------------------------------------------+
| Should you take slow release protein before going to bed? | Slow release protein is the perfect solution to the       |
|                                                           | trainer’s problem - there’s simply not enough hours in    |
|                                                           | the day. Take some slow release protein before going to   |
|                                                           | sleep, and you can be sure that as you rest, your body is |
|                                                           | still absorbing protein.                                  |
+-----------------------------------------------------------+-----------------------------------------------------------+
| Who should take slow release protein powders?             | Slow release protein powders are suitable for anyone, and |
|                                                           | we have a range of products that will work for you,       |
|                                                           | whether you’re following a gluten-free or vegetarian      |
|                                                           | diet.                                                     |
+-----------------------------------------------------------+-----------------------------------------------------------+
| Are slow release protein powders vegan?                   | Unfortunately, because slow release protein powders       |
|                                                           | contain Micellar Casein, which is a milk product, they    |
|                                                           | are unsuitable for a vegan diet.                          |
